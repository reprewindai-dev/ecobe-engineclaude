# ECOBE Lead Generation Demo

A simple demo that shows how ECOBE Engine optimizes lead generation searches based on carbon footprint.

## Setup

1. Update `.env.local` with your ECOBE Engine URL and API key
2. Install dependencies: `npm install`
3. Run locally: `npm run dev`
4. Build for production: `npm run build`

## Features

- **Real API Calls**: Uses actual ECOBE Engine optimization API
- **Any Location**: Search for businesses in Vancouver, Toronto, Austin, anywhere
- **Carbon Optimization**: Shows carbon savings from green routing
- **No Authentication**: Open demo for anyone to test

## Usage

Enter a business type (e.g., "SaaS companies") and location (e.g., "Vancouver BC") to find businesses with carbon-optimized search.
